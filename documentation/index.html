MonkeyDB Documentation

Overview Features Installation Usage API Reference

Overview

MonkeyDB is a lightweight, pure Python database engine with basic SQL-like functionality. It is designed for simplicity and ease of use, making it ideal for small projects and educational purposes.

Features

  • Create tables with specified columns and data types.
  • Insert records into tables with automatic ID generation.
  • Select and filter records based on conditions.
  • JSON-based storage for easy data management.
  • Thread-safe operations for concurrent access.

Installation

To install MonkeyDB, follow these steps:

pip install monkeydb

Usage

Step-by-Step Guide to Use MonkeyDB

  1. Make a new folder for your project.
  2. In the same directory as your root Python file, create a new Python file named monkeyDB.py.
  3. Include MonkeyDB in your Python file using the following import statement:
    4. from monkeydb import DatabaseEngine
  4. Initialize the database engine:
    5. db = DatabaseEngine()
  5. Create a table:
    6. db.execute_query("CREATE TABLE users (id INTEGER, name TEXT, age INTEGER)")
  6. Insert records into the table:
    7. db.execute_query("INSERT INTO users (id, name, age) VALUES (1, 'Alice', 30)")
  7. Select records from the table:
    8. results = db.execute_query("SELECT * FROM users WHERE age = 30")
  8. Print the results:
    9. print(results)

Example Code


# monkeyDB.py
from monkeydb import DatabaseEngine

# Initialize the database engine
db = DatabaseEngine()

# Create a table
db.execute_query("CREATE TABLE users (id INTEGER, name TEXT, age INTEGER)")

# Insert records
db.execute_query("INSERT INTO users (id, name, age) VALUES (1, 'Alice', 30)")
db.execute_query("INSERT INTO users (id, name, age) VALUES (2, 'Bob', 25)")

# Select records
results = db.execute_query("SELECT * FROM users WHERE age = 30")
print(results)

API Reference

StorageEngine

Handles the storage of data in JSON files.

  • create_table_file(table_name: str) -> str: Creates a new table file.
  • write_data(table_name: str, data: List[Dict[str, Any]]): Writes data to a table file.
  • read_data(table_name: str) -> List[Dict[str, Any]]: Reads data from a table file.

QueryParser

Parses SQL-like queries.

  • parse_create_table(query: str) -> Dict[str, Any]: Parses CREATE TABLE queries.
  • parse_insert(query: str) -> Dict[str, Any]: Parses INSERT queries.
  • parse_select(query: str) -> Dict[str, Any]: Parses SELECT queries.

DatabaseEngine

Handles the execution of queries and manages tables.

  • execute_query(query: str) -> Union[List[Dict], None]: Executes a SQL-like query.
  • _create_table(parsed_query: Dict[str, Any]): Creates a new table.
  • _insert(parsed_query: Dict[str, Any]): Inserts data into a table.
  • _select(parsed_query: Dict[str, Any]): Selects data from a table.
